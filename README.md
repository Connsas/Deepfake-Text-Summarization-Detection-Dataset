# Deepfake Text Summarization Detection Dataset 🚀

This repository contains a high-quality, large-scale dataset specifically designed for the detection of **AI-generated (Deepfake) text summaries**. The dataset focuses on distinguishing between human-written academic summaries and those generated by state-of-the-art **Large Language Models (LLMs)**.

## 📌 Overview

The rapid advancement of LLMs has made it increasingly difficult to differentiate synthetic content from human writing, especially in structured and formal contexts like academic summarization. This dataset was developed to move beyond superficial artifact-based detection (HTML, markdown tags, etc.) toward **deep linguistic and structural pattern recognition**.

## ✨ Key Features

- **Large-Scale Data:** A total of **63,000 samples** (balanced between human and AI).
- **Multi-Model Diversity:** Includes synthetic summaries generated by:
  - `Llama-3.1`
  - `Gemma-2`
  - `Qwen-2.5`
  - `Phi-3`
  - `Falcon`
  - `Mistral`
- **Zero-Artifact Strategy:** The dataset is curated to ensure models learn from **contextual and semantic patterns** rather than technical artifacts or formatting cues.
- **Academic Focus:** Specifically addresses the "False Positive" challenge in formal and prescriptive human writing.

## 📊 Dataset Structure

The data is typically structured as follows:

| Column | Description |
| :--- | :--- |
| `text` | The full content of the summary. |
| `label` | `0` for Human-written, `1` for AI-generated. |
| `model_name` | The architecture used for generation (or 'human'). |
